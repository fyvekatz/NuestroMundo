Simple Style3 example with Home Screen SDK2.0

1.Description

This is a Style3 simple sample that uses Brightscript and Roku Scenegraph (RSG) to create a Roku channel.

Concepts used: Channel manifest, Scenegraph nodes, APIs, XML, and scenes; renderable nodes,

2. Pre-requisites

Before going over the sample, the reader is encouraged to get familiar with:
- Brightscript
- Roku Scenegraph (RSG)
- How to setup a project

3. Base Hierarchy of files in project

- Home scene
    - Poster overhang s3 logo and Roku logo
    - Home screen
        - Label id="title" and id="description"
        - Rectangle
